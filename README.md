# Web Services with Spring Boot and JPA / Hibernate

Project developed during the "Web Services with Spring Boot and JPA / Hibernate" section of the Java Complete Course.

## Project Objective

The objective of this project is to practice the development of a REST API using Spring Boot, applying concepts such as:

- Layered architecture
- Complete CRUD operations
- JPA / Hibernate
- H2 Database
- Entity relationships
- Exception handling
- Data persistence

## Technologies Used

- Java
- Spring Boot
- Spring Data JPA
- Hibernate
- Maven
- H2 Database
- PostgreSQL

## Features

- User registration
- User retrieval
- User update
- User deletion
- Order registration
- Entity relationships
- Custom exception handling

## Project Structure

The project was organized using a layered architecture:

- Resource
- Service
- Repository
- Entities
- Exceptions

## Domain Model

The project works with the following entities:

- User
- Order
- Product
- Category
- OrderItem
- Payment

# Web Services with Spring Boot and JPA / Hibernate

Projeto desenvolvido durante a seção "Projeto web services com Spring Boot e JPA / Hibernate" do curso Java Completo.

## Objetivo do Projeto

O objetivo deste projeto é praticar o desenvolvimento de uma API REST utilizando Spring Boot, aplicando conceitos como:

- Arquitetura em camadas
- CRUD completo
- JPA / Hibernate
- Banco de dados H2
- Relacionamentos entre entidades
- Tratamento de exceções
- Persistência de dados

## Tecnologias Utilizadas

- Java
- Spring Boot
- Spring Data JPA
- Hibernate
- Maven
- H2 Database
- PostgreSQL

## Funcionalidades

- Cadastro de usuários
- Busca de usuários
- Atualização de usuários
- Remoção de usuários
- Cadastro de pedidos
- Relacionamento entre entidades
- Tratamento de exceções personalizadas

## Estrutura do Projeto

O projeto foi organizado utilizando arquitetura em camadas:

- Resource
- Service
- Repository
- Entities
- Exceptions

## Modelo de Domínio

O projeto trabalha com as seguintes entidades:

- User
- Order

- Payment
